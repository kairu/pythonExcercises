# Python Exercises

This repository contains a collection of Python exercises covering various programming concepts and challenges. These exercises are designed to help improve Python programming skills through practical implementation.

## Exercises Included

1. **Exercise1.py**: Age calculator that determines when a user will turn 100 years old.
2. **Exercise2.py**: Odd/Even number checker with additional features for multiples of 4 and division checks.
3. **Exercise3.py**: List manipulation exercise to filter elements based on user input.
4. **Exercise4.py**: Program to find divisors of a given number.
5. **Exercise5.py**: List comparison exercise to find common elements between two lists.
6. **RegexPractice.py**: Regular expression practice with XML-like data.

## Additional Files

- **Hello Test.py**: A simple script demonstrating basic Python syntax.

## Getting Started

To run these exercises, make sure you have Python installed on your system. Clone this repository and run each exercise file individually using a Python interpreter.
